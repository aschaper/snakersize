# snakersize

Put on your sweatband and start snakersizing!

## Installation

```ruby
gem install snakersize
```

## Usage

Give your strings the snakersize they deserve!

```ruby
require 'snakersize'

"A boring string".snakersize
#=> "a_boring_string"
```

What a workout! Don't wait, snakersize your strings today!
